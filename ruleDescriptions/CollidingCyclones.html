Colliding Cyclones

This CA produces spiraling cyclones.

Each cell can only update its state from n to n+1 (modulo N) where N is the number of states. In other words, each cell cycles through all possible values. Also, a cell only increments its current state (n) if there are 3 or more neighbors with the next state (n+1).

Cyclic CA such as this one were explored and popularized by David Griffeath.
