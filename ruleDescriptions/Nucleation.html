Creeping Nucleation (Spirals)

Colorful spirals nucleate out of initial disorder. The time it takes to nucleate depends on the number of states, with smaller numbers nucleating faster. For the default 14 states, run the simulation for 250 or more generations to see the spirals.

This rule is a classic cyclic CA. Each cell can only update its state from n to n+1 (modulo N) where N is the number of states. In other words, each cell cycles through all possible values. Also, a cell only increments its current state (n) if there is 1 or more neighbors with the next state (n+1).
