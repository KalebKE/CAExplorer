Getting Started

Running a simulation is as easy as pressing the start button.  But if you would like to change the simulation, follow these easy steps.  Note that tooltip explanations are available for all options.

Four easy steps:

  1. Select a rule on the Rule tab.

  2. Select properties on the Properties tab.

  3. Select initial conditions on the Initial State tab.

  4. Submit. Then run the simulation from the Controls tab.

That's it! And you don't even have to do these in the given order. Mix it up. You may submit and run at any time.

 

Want a few more details?

(1) Select a rule on the Rule tab.

  • Optional details:

  • After selecting (with a single click), you can submit and run, or you can wait to submit and proceed to step 2.

  • Tooltips tell you what each rule can do.

  • Select a rule to see an expanded description displayed at the bottom of the tab.

  • Double-clicking will submit the rule.

  • If available, select a rule number (a tooltip tells you the range of available choices).

  • If your desired rule is "grayed out" (unavailable), then a tooltip tells you what lattice must be selected to make the rule available (see step 2).

(2) Select properties on the Properties tab.

  • Optional details:

  • You can submit and run, or you can wait to submit and proceed to step 3.

  • The "For best results..." description can help you make good choices.

  • The most common lattices are highlighted in light blue, and the next most common lattices are marked in dark blue.  Other lattices are rarely used.

  • Rows and columns determine the lattice size.  Start small (50 X 50), then try larger if desired.  Some simulations are slow on large lattices.

  • Small numbers of "States" like 2 or 3 are most common.

  • Some rules have "More Properties" that can be selected with the button at the bottom.

(3) Select initial conditions on the Initial State tab.

  • Optional details:

  • The most common initial conditions are at the top of the tab.

(4) Submit. Then run the simulation from the Controls tab.

  • Optional details:

  • Enjoy.
